# กติกาพฤติกรรมของ Agent (ภาษาไทย)

## ภาษาในการสื่อสาร
- Agent ต้องสื่อสารเป็นภาษาไทยเป็นหลัก
- ห้ามเปลี่ยนไปใช้ภาษาอังกฤษเอง
- ถ้าคำถามหรือคำสั่งคลุมเครือ:
  - ต้องถามกลับเพื่อขอความชัดเจนก่อน
  - ห้ามเดาเอง

---

## การทำงานกับ Terminal / CLI
- ห้ามรันคำสั่งใด ๆ ใน terminal แทนผู้ใช้
- ถ้าจำเป็นต้องใช้ terminal:
  - ให้แสดงคำสั่งทั้งหมดเป็นข้อความ
  - ผู้ใช้เป็นคนคัดลอกไปวางและรันเองเท่านั้น
- ห้ามใช้คำสั่งที่มีผลทำลายระบบโดยไม่ขออนุญาต
- ใช้ pnpm เป็นหลัก

## การตั้งชื่อ ไฟล ตัวแปร หรือ ชื่อหน้า
- พยายามตั้งแบบนี้ก่อน เช่น scan_json use_index_db
- ถ้าจำเป็นต้องใช้ - หรือ ตัวพิมใหญ่ ให้ถามก่อน
- อย่าตั้งชื่อเองตามใจชอบ ให้ถามก่อน ก่อนถามให้เสนอชื่อมาก่อนโดยพยายามให้เหมือนชื่อตัวแปรที่เกี่ยวข้องมากที่สุด หรือถ้าตัวแปรเดียวกันก็ชื่อเดิม

ตัวอย่างที่ถูกต้อง:
```bash
pnpm install
pnpm run dev
